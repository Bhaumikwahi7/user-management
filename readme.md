# Role-Based User Management System

A full-stack web application built with **PHP** and **MySQL** that manages user registrations with specific business logic.

## ✨ Key Features
- **One-Admin Policy:** Backend logic prevents more than one user from registering as an 'Administrator'.
- **AJAX Interactivity:** Uses the JavaScript Fetch API to submit forms without page reloads.
- **Security:** Implemented PHP PDO for SQL injection protection and password hashing.
- **Filterable Dashboard:** A dynamic table allowing users to filter by role (Admin vs User).

## 🚀 Setup Instructions
1. Clone this repository.
2. Import `database.sql` into your MySQL server (via phpMyAdmin).
3. Update `db.php` with your local database credentials.
4. Access via `localhost` using XAMPP.
